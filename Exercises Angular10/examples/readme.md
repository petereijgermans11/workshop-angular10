# Example projects Training Angular

Every folder is a seperate project: use `npm install`  and `npm start` from a command prompt. 

## Contents
- AAA - Hello World, Data binding examples
- Cxx - Services
- Dxx - Components
- Exx - Router examples
- Fxx - Forms
- Gxx - Miscellaneous
- Hxx - Work in Progress (wip)
